# React ui lib

Template for react + tailwindcss ui library

## Local developpement

The library and the projects which use it must be locally placed in a monorepo.
The folder structure looks like that :

```
parent-folder/
├── packages/
│   ├── react-ui-lib/
├── sites/
│   └── my-super-app/
├── package.json
└── package-lock.json
```
